# Simulation Requirement

จำลองกระแสสังคมของโครงการ AI Pass TH / TH-AI Passport ในประเทศไทย เป็นระยะเวลา 90 วันหลังจากโครงการเปิดตัว

ต้องการดู:
1. กระแสสนับสนุนและคัดค้านจะพัฒนาอย่างไร
2. กลุ่ม stakeholder ใดจะเป็นตัวเร่งกระแส
3. จุดเสี่ยงที่ทำให้โครงการถูกมองว่าไม่คุ้มค่า
4. เงื่อนไขที่ทำให้โครงการถูกมองว่าสำเร็จ
5. narrative ที่รัฐบาลควรสื่อสารเพื่อลดแรงต้าน
6. KPI ที่ควรเปิดเผยเพื่อสร้างความน่าเชื่อถือ
7. scenario ที่เป็นไปได้ 3 แบบ:
   - Best case
   - Base case
   - Worst case

ให้ simulation เน้น public opinion, policy risk, adoption behavior, media amplification, technical criticism และ trust recovery strategy
